## Consignes 

Réalisez un tirage de loto de 7 chiffres aléatoires. Pour cela, il vous faudra procéder en 4 étapes:

1. Créez un tableau de 7 éléments 

2. Alimentez ce tableau avec des nombres aléatoires, tous différents entre eux et compris entre 1 et 49 inclus. Il faudra donc créer une boucle qui tire un nombre et l'ajoute dans le tableau seulement s'il n'a pas déjà été tiré.

3. Affichez le contenu du tableau à l'aide d'une méthode une fois que le tirage des 7 nombres est terminé.

4. Insérez votre code dans un fichier Program.cs.

## Vérifiez-bien que vous avez les éléments suivants :

* Il y a un tableau d'entiers déclaré avec int[] ;
* Le tableau a une taille allouée de 7 éléments ;
* Il y a une boucle qui permet d'alimenter le tableau ;
* Un nombre aléatoire est tiré à chaque tour de boucle ;
* Ce nombre aléatoire est ajouté dans le tableau s'il n'a pas déjà été tiré ;
* Il y a une méthode qui renvoie un booléen ;
* Cette méthode parcourt le tableau et vérifie que le nombre tiré n'est pas déjà présent dans le tableau ;
* Si le nombre est déjà présent dans le tableau, on sort de la boucle ;
* La méthode ne parcourt pas les cases du tableau qui n'ont pas encore été remplies ;
* Chaque élément du tableau est affiché ;
* Une boucle est utilisée pour parcourir le tableau ;
* Les éléments du tableau sont tous différents les uns des autres.
